# NeurIPS Paper (Anonymous Submission)

- Entry file: `main.tex`
- Style file: `neurips_2024.sty`
- Bibliography: `references.bib`

To compile in Overleaf:
- Create a new project and upload this folder’s contents.
- Set the main file to `main.tex`.

To compile locally:
- Run `make pdf` (uses `latexmk` if available, otherwise falls back to `pdflatex` + `bibtex`).
- Output: `main.pdf`
